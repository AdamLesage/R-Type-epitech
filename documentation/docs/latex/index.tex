There are two ways to install and run the project, depending on whether you are developing or using the software as a regular user.\hypertarget{index_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{For Developers (Development Mode)}{For Developers (Development Mode)}}\label{index_autotoc_md2}
To install the project in development mode, you can clone the repository and compile it with the {\ttfamily Makefile}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ git@github.com:AdamLesage/R-\/Type-\/epitech.git}
\DoxyCodeLine{cd\ R-\/Type-\/epitech}
\DoxyCodeLine{make}

\end{DoxyCode}


This will create an executable file called {\ttfamily rtype} in the {\ttfamily build} directory.\hypertarget{index_autotoc_md3}{}\doxysubsubsection{\texorpdfstring{For Users (Production Mode)}{For Users (Production Mode)}}\label{index_autotoc_md3}
If you want to install and use the software as a regular user, simply download the latest release from the \href{https://github.com/AdamLesage/R-Type-epitech/releases}{\texttt{ Releases}} section on Git\+Hub.

For {\bfseries{Linux}} users\+:
\begin{DoxyEnumerate}
\item Download the Linux release from the release page.
\item Make the installer executable\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{chmod\ +x\ linux\_installer.sh}

\end{DoxyCode}

\item Run the installer\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{./linux\_installer.sh}

\end{DoxyCode}

\end{DoxyEnumerate}

For more details on the installation, check the documentation file\+: {\ttfamily ./documentation/\+LINUX\+\_\+\+INSTALL.md}.

For {\bfseries{Windows}} users\+:


\begin{DoxyEnumerate}
\item Download the Windows release from the release page.
\item Right click on the windows\+\_\+installer.\+exe and run it as an administrator

\item The project will build and will be ready to use
